# Algorithmic-Music-Chord-Generator
Hardware-Accelerated Chord Progression GeneratorObjective: A Verilog-based engine that generates musically coherent chord progressions using a Finite State Machine (FSM).Technical Evidence:
* Music Theory Logic: Hard-coded Diatonic scale constraints into a Moore FSM to ensure harmonic stability.
* Connectivity: Real-time MIDI data transmission via UART, compatible with DAWs like FL Studio.
* Work Sample: Included a sample MIDI file in the repository to prove the algorithm's output quality.
